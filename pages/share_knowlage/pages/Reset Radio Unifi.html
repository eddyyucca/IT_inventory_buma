Reset Radio Unifi

PT. BUKITMAKMUR-IPR

#BeraniBeda"

____________________________________________________________________________________________________________________________

  1. Buka aplikasi SSH, contoh Putty. Sesuaikan ip dengan ip perangkat anda.
     
  2. Masukkan default username password, jika AP masih baru defaultnya username : ubnt, password ubnt. jika sebelumnya sudah pernah melakukan addopt dan setup maka bisa menggunakan username password yang ada di controller unifi,                                        
  3. Paste script berikut syswrapper.sh restore-default (abaikan jika muncul popup error, ini menandakan jika sudah berhasil ter reset)
  4. Done
         setelah acces point ter reset (kembali ke kondisi default) maka sudah bisa di addopt, dan setting kembali menggunakan uifi controller.
